# Smart File Manager

A clean, minimal file management system built with Python and Streamlit.

![Python](https://img.shields.io/badge/Python-3.13-blue?style=flat-square&logo=python)
![Streamlit](https://img.shields.io/badge/Streamlit-1.58.0-red?style=flat-square&logo=streamlit)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

---

## Features

- **Create** — Create new files with custom content or upload from disk
- **Read** — View file contents with syntax display and download option
- **Update** — Rename, append to, or overwrite existing files
- **Delete** — Safely delete files with confirmation step

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/smart-file-manager.git
cd smart-file-manager
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.
